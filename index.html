Que hacemos

GetApp es una app mobile para poder tener una alternativa cuando no podemos recibir alguna paqueteria por cualquier tema personal, con un sistema de dos tipos de usuarios, uno que pueda reservar su hogar y tiempo para recibir productos y para el que busca al otro tipo de usuario asi tener su producto.
